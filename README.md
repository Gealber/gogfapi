A Go language wrapper around gfapi.

To use in your Go program get/download the package with,  
``$ go get github.com/kshlm/gogfapi``
and import as,  
``import "github.com/kshlm/gogfapi/gfapi"``

and get/download the package with,  
``$ go get github.com/kshlm/gogfapi``
